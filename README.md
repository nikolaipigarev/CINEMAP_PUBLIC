# CINEMAP XML Catalog

Публичный XML каталог для импорта на сайт.

## Файлы

- `CINEMAP_YOURRENTAL_CATALOG_enriched_llm.xml` - основной XML файл каталога

## Прямая ссылка для скачивания

После публикации на GitHub Pages будет доступна по ссылке:
`https://[username].github.io/CINEMAP_AI/public_xml/CINEMAP_YOURRENTAL_CATALOG_enriched_llm.xml`

## Как опубликовать

1. Создать новый репозиторий на GitHub
2. Включить GitHub Pages в настройках репозитория
3. Выбрать источник `main` ветка и `/` папка
4. Загрузить файлы из папки `public_xml`
